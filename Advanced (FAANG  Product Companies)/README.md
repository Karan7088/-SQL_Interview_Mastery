# 🚀 Advanced SQL Interview Questions (FAANG / Product Companies)

## 📌 Overview

This repository contains a curated set of **advanced-level SQL interview questions** commonly asked in **FAANG and top product-based companies**.

These problems focus on **real-world data scenarios**, testing not just SQL syntax but also **analytical thinking, edge case handling, and business understanding**.

---

## 🧠 Problem List

### 🔴 Advanced SQL Scenarios

1. **Retention Analysis (Day 1, Day 7 Users)**
   Analyze user engagement by calculating how many users return after specific time intervals.

2. **Funnel Analysis (Signup → Purchase → Repeat)**
   Track user journey across multiple stages and measure conversion rates.

3. **Find Churned Users**
   Identify users who stopped engaging with the platform after a certain period.

4. **Calculate DAU / WAU / MAU**
   Compute Daily, Weekly, and Monthly Active Users for engagement tracking.

5. **Detect Fraud Transactions (Pattern-Based)**
   Identify suspicious patterns like rapid transactions, duplicates, or anomalies.

6. **Ranking with Ties (`RANK` vs `DENSE_RANK`)**
   Understand ranking behavior when duplicate values exist.

7. **Find Median Salary**
   Solve using window functions and handle even/odd row scenarios.

8. **Event Stream Problem (Start, Pause, Resume, Cancel)**
   Track state transitions in event-based systems.

9. **Gap & Island Problems (Continuous Sequences)**
   Identify continuous vs broken sequences in time-series data.

10. **Build Cohort Analysis Table**
    Group users by signup period and track retention over time.
